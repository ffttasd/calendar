# 更新日志

## 2024-03-04 - TODO视图切换功能

### 新增功能
- ✨ 添加了TODO视图切换功能
- 📋 默认显示简洁的TODO列表模式
- 🎯 可手动切换到四象限管理模式
- 💾 视图模式偏好会自动保存
- 🖱️ 简洁模式支持点击区域添加任务
- 🗑️ 简洁模式支持hover显示删除按钮
- 🎯 简洁模式支持拖拽TODO到日程

### 功能说明

#### 简洁模式（默认）
- 单一列表显示所有任务
- 适合快速记录和查看任务
- 界面更加简洁清爽
- 点击空白区域即可添加新任务
- hover显示删除按钮
- 支持拖拽任务到日程表创建事件

#### 四象限模式
- 按重要性和紧急性分类任务
- 帮助进行时间管理和优先级排序
- 四个象限：
  - 重要且紧急
  - 重要但不紧急
  - 不重要但紧急
  - 不重要且不紧急
- 支持拖拽任务到日程表创建事件

### 使用方法
1. 在右侧边栏的"任务管理"区域，点击右上角的切换按钮
2. 系统会在简洁模式和四象限模式之间切换
3. 切换后的视图偏好会自动保存
4. 点击任务区域空白处添加新任务
5. 拖拽任务到左侧日程表可快速创建事件

### 技术实现
- 新增 `todoViewMode` 状态管理（'simple' | 'quadrant'）
- 使用 localStorage 持久化视图偏好
- 支持两种模式的独立数据存储
- 导入导出功能兼容新的数据结构
- 简洁模式和四象限模式共享拖拽逻辑
- 统一的交互体验（点击添加、hover删除、拖拽创建）

### 数据结构
```javascript
todos: {
  simple: [],  // 简洁模式的任务列表
  1: [],       // 四象限：重要且紧急
  2: [],       // 四象限：重要但不紧急
  3: [],       // 四象限：不重要但紧急
  4: []        // 四象限：不重要且不紧急
}
```
